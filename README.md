Instructions:

1. Clone the repository 
```bash
$ git clone https://github.com/kevinyvv/memoir
```
2. Install required frontend dependencies
```bash
$ cd front/memoir
$ npm install
```
4. Install required backend dependencies
```bash
$ cd backend
$ pip install requirements.txt
```
5. Run Flask server
```bash
$ python server.py
```
5. Run React server
```bash
$ npm run start 
```
